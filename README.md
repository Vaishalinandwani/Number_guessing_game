Number Guessing Game

Overview
This is a simple Java console application that generates a random number between 1 and 100. The user has to guess the correct number, and the program provides hints if the guess is too high or too low until the correct number is guessed.

Features
* Random number generation
* User input using Scanner
* High/Low hints
* Counts the number of attempts
* Runs until the correct guess is made

Technologies Used
* Java
* Random Class
* Scanner Class

How to Run
1. Clone this repository.
2. Compile the Java file.
3. Run the program.
4. Enter your guesses in the console.

Concepts Used
* Loops
* Conditional statements
* Random number generation
* User input handling

Future Improvements
* Multiple rounds
* Difficulty levels
* Score system
* Limited attempts

Learning Outcome
This project helped strengthen my understanding of Java programming by implementing random number generation, loops, conditionals, and user interaction in a real-world console application.
